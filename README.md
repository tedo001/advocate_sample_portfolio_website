# ⚖️ Advocate Portfolio Website

A modern, responsive portfolio website designed for legal professionals.  
Built using HTML, CSS, and JavaScript, this project showcases a professional advocate profile with services, case records, and contact features.

---

## 🌐 Live Demo
👉 https://sampleadvocatewebsite.vercel.app

---

## ✨ Features

- 🧑‍⚖️ Professional hero section
- 📊 Experience & achievements stats
- 📚 Practice areas display
- 📄 Case history section
- 📬 Contact / enquiry form
- 📱 Fully responsive design
- 🎨 Clean and modern UI
